## Text Model Module Implementation

This folder contains the implementation of the text model component for the multimodal emotion recognition model. The text model supports the following models:
- XMLRoBERTa
- mBart

## Supported Models

### XMLRoBERTa
The `XMLRoBERTaModelWrapper` class wraps the XMLRoBERTa model from the Hugging Face Transformers library.

### mBart
The `MBartModelWrapper` class wraps the mBart model from the Hugging Face Transformers library.
